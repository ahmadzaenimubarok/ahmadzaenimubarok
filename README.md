PHP and Python backend developer from Semarang, Indonesia, focused on **API integration**, **workflow automation**, and connecting systems to AI tools like **Claude**, **OpenAI**, and **LangChain**.

Most of my clients have a working product that needs *more* — more integrations, smarter automation, or an AI layer they've been putting off. I map the system before writing a line, identify the fastest path to what you need, flag scope issues early, and deliver working, documented code.
